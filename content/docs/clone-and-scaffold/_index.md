가장 널리 쓰이는 한 줄 명령형 스킬은 "무언가를 빠르게 만들어내는" 스킬입니다. 특정 사이트 디자인을 그대로 클론 코딩하거나, 프로젝트 뼈대(보일러플레이트)를 즉시 생성하거나, 디자인 목업을 코드로 바꾸는 작업이 여기 속합니다.

{{< callout type="warning" >}}
클론·생성 스킬은 "빨리 만든다"에만 집중하면 저작권·라이선스 문제나 접근성 누락으로 이어지기 쉽습니다. 이 섹션의 각 페이지는 속도뿐 아니라 무엇을 확인해야 안전하게 재사용할 수 있는지도 함께 다룹니다.
{{< /callout >}}

## 이 섹션에서 다루는 내용

1. **사이트 디자인 한 번에 클론 코딩** — 대상 사이트 분석부터 코드 생성, 검증까지
2. **보일러플레이트/스캐폴딩 자동 생성** — 프로젝트 뼈대를 스킬로 표준화하기
3. **디자인 → 코드 자동화** — Figma 등 디자인 목업을 코드로 변환하는 스킬
