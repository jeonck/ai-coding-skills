개별 스킬을 하나씩 손으로 부르는 단계를 넘어서면, 여러 스킬을 **병렬로, 순서대로, 혹은 이벤트에 반응해** 자동으로 연결하는 방법이 필요합니다. 이 섹션은 멀티에이전트 병렬 실행, 훅 자동화, MCP를 통한 외부 도구 연동을 다룹니다.

{{< callout type="warning" >}}
여러 에이전트나 훅을 동시에 돌리기 시작하면 "누가 무엇을 언제 실행했는지" 추적이 어려워집니다. 오케스트레이션을 도입하기 전에 각 단위 스킬이 [기반 개념](../foundations/)에서 다룬 것처럼 명확한 트리거 조건과 결과 보고 형식을 갖추고 있는지 먼저 확인하세요.
{{< /callout >}}

## 이 섹션에서 다루는 내용

1. **멀티에이전트 병렬 실행** — 여러 서브에이전트를 동시에 돌려 시간을 단축하는 법
2. **훅으로 반복 작업 자동화** — 커밋·배포 시점마다 무조건 실행되어야 하는 작업 연결하기
3. **MCP 연동으로 외부 도구 활용** — 브라우저, 디자인 툴, 사내 시스템과 연결하기
