## 한 줄 명령

오른쪽 위 복사 아이콘을 눌러 그대로 붙여넣으세요.

```text
이 사이트(URL)처럼 만들어줘, 반응형까지 맞춰서
```

## 무엇이 일어나는가

1. 브라우저 자동화로 대상 URL의 DOM·스타일시트·스크린샷을 수집
2. 색상·타이포그래피·간격을 재사용 가능한 변수로 추출
3. 헤더/히어로/카드/푸터 단위로 컴포넌트 분해
4. 목표 스택(React, Next.js, 정적 HTML 등)으로 코드 생성
5. 로컬 빌드 후 원본과 스크린샷을 나란히 비교 검증

## 사전 조건

- 클론 대상이 본인 소유이거나 학습·데모 목적임이 명확할 것
- 브라우저 자동화 도구(예: Claude Code의 `claude-in-chrome`) 사용 가능
- 목표 프레임워크의 로컬 빌드 환경
