## Claude Code

- **링크**: https://claude.com/product/claude-code
- **설명**: 터미널 기반 AI 코딩 에이전트. 스킬, 서브에이전트, 훅, MCP 서버 연동을 통해 이 로드맵에서 다루는 대부분의 자동화를 실행할 수 있습니다.
- **설치**: `npm install -g @anthropic-ai/claude-code`

## Claude Agent SDK

- **링크**: https://docs.claude.com/en/api/agent-sdk
- **설명**: Claude Code 내부에서 쓰이는 것과 동일한 에이전트 하네스를 자체 애플리케이션에 임베드할 때 사용하는 SDK입니다.
- **설치**: `pip install claude-agent-sdk` 또는 `npm install @anthropic-ai/claude-agent-sdk`

## Model Context Protocol (MCP) SDK

- **링크**: https://modelcontextprotocol.io/
- **설명**: 에이전트가 브라우저, 디자인 툴, 사내 시스템 등 외부 도구를 표준화된 방식으로 호출하도록 만드는 프로토콜/SDK입니다.
- **설치**: `npm install @modelcontextprotocol/sdk`
